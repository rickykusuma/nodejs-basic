# Node.js Basic Tutorial

This repository contains simple source code examples for learning **Node.js fundamentals**.

## Contents
The examples cover basic concepts such as:
- Node.js runtime basics
- Modules and `require`
- Asynchronous code
- Events
- File system (FS)
- HTTP server and client
- Process and standard library usage
- Worker threads

## Purpose
This repository is created as a **learning log and practice project** while studying Node.js basics.
Each file focuses on one concept to keep the examples small and easy to understand.

## How to Use
1. Install Node.js (LTS version recommended)
2. Clone this repository
3. Run any file using:
   ```bash
   node filename.js
   ```

## Notes
- This project is for learning purposes
- The code is intentionally simple and not production-ready
